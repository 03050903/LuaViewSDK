Class VScrollView

VScrollView 垂直滚动的View

VScrollView的父类是View, 子类有TableView, CollectionView

全局函数

VScrollView () 通过这个方法创建一个VScrollView对象

实例方法

contentSize (w, h) 设置ContentSize大小
contentSize () 获取ContentSize大小
offset (x, y, animation) 设置ContentSize大小
offset () 获取contentOffset大小
contentInset (top, left, bottom, right) 设置contentInset(只对IOS有效)
contentInset () 获取contentInset(只对IOS有效)
showScrollIndicator (showsHorizontal) 设置是否显示滚动条
showScrollIndicator () 获取是否显示滚动条信息


全局函数

VScrollView ()
通过这个方法创建一个VScrollView对象

返回值:

    VScrollView view

例子:

     Cell
 local scrW, scrH = System.screenSize();
 local cellW = scrW;
 local cellH = 130;
 CellJinrishouxuan = {};
 CellJinrishouxuan.width = cellW;
 CellJinrishouxuan.height = cellH;
 function CellJinrishouxuan.create(originCell)
     local cell = {};
     cell.cellInited = false;
     function cell.init()
         if (self.cellInited == false ) then
             self.cellInited = true;
             self.scrollView = VScrollView();
             self.scrollView.frame( 0, 0, cellW, cellH );
             self.items = {};
             for i=1,3 do
             	local item = CellProductSmall.create(nil, cellW / 3, cellH );
             	self.scrollView.addView(item.view);
             	item.xy(cellH*(i-1),0 );
             	self.items[i] = item;
             	item.bizIconUrl = "https://gw.alicdn.com/tps/TB1Q6oxKpXXXXbcXVXXXXXXXXXX-86-96.png";
             end
             self.scrollView.contentSize(3*cellH, cellH);
             self.scrollView.showScrollIndicator(true,false)
             self.scrollView.backgroundColor(0xffffff);
         end
     end
     function cell.layout(model,option)
 		if(not model) then return end
     	local itemList = model.itemList;
     	local num = table:getn(itemList);
     	for i=1, num do
     		local model = itemList[i];
     		local item  = self.items[i];
     		item.layout(model,option);
     	end
     end
     cell.init();
     return cell;
 end

实例方法

contentSize (w, h)
设置ContentSize大小

参数:

  • w number 宽
  • h number 高
contentSize ()
获取ContentSize大小

返回值:

  1. number w 宽
  2. number h 高
offset (x, y, animation)
设置ContentSize大小

参数:

  • x number x滚动偏移位置
  • y number y滚动偏移位置
  • animation bool 是否动画 参数可以省略
offset ()
获取contentOffset大小

返回值:

  1. number x
  2. number y
contentInset (top, left, bottom, right)
设置contentInset(只对IOS有效)

参数:

  • top number
  • left number
  • bottom number
  • right number
contentInset ()
获取contentInset(只对IOS有效)

返回值:

  1. number top
  2. number left
  3. number bottom
  4. number right
showScrollIndicator (showsHorizontal)
设置是否显示滚动条

参数:

  • showsHorizontal bool 显示水平滚动条
showScrollIndicator ()
获取是否显示滚动条信息

返回值:

    bool showsHorizontal 显示水平滚动条
