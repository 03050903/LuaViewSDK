__index (index)
读取值

参数:

  • index int 索引号

例子:

     local data = Data("123456");
 print(data[2]);
__newindex (index)
赋值

参数:

  • index int 索引号

     local data = Data("123456");

 print(data[2]);

参数:

  • data Data 数组

返回值:

    bool 是否相同

例子:

     local data1 = Data('201508271013' );
 local data2 = Data('测试');
 print(data1 + data2);
